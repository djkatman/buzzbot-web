# X BuzzBot Web - スマホ対応版

バズ投稿のURLまたはテキストを貼り付けてAIで投稿文を生成するWebアプリ。
PCでもスマホでも使えます。

## デプロイ手順（Vercel）

1. GitHubにこのフォルダをリポジトリとして push する
2. vercel.com でアカウント作成（無料）
3. 「New Project」→ GitHubリポジトリを選択
4. そのままDeployをクリック
5. 発行されたURL（例: buzzbot-web.vercel.app）にアクセス

## 使い方

1. ブラウザでアクセス
2. 「⚙️ 設定」タブでClaude APIキーを入力して保存
3. 「✨ AI生成」タブでバズ投稿のURLまたはテキストを貼り付け
4. 「AI生成する」ボタンをクリック
5. 生成された3パターンからコピーまたは引用投稿

## スマホでの使い方

1. XでバズってるポストをタップSH → 「共有」→「リンクをコピー」
2. ブラウザでこのアプリを開く
3. URLを貼り付けて生成
4. コピーしてXアプリで投稿

## 費用

- ホスティング（Vercel）: 無料
